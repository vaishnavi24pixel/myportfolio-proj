## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/vaishnavi24pixel/myportfolio-proj.git
   ```
2. Navigate to the project directory:
   ```bash
   cd myportfolio-proj
   ```
3. Open `index.html` in your browser to view the portfolio.
